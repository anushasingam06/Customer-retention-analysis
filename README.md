# Customer Retention Analysis — E-commerce

## 📌 Project Overview
A Business Analyst project analyzing customer buying behavior on an e-commerce platform to identify at-risk customers and define requirements for a retention strategy.

## 🎯 Objectives
- Identify one-time buyers who never returned
- Analyze review scores by product category
- Segment customers by city and purchase frequency
- Define system requirements for a retention alert system

## 🛠️ Tools Used
- SQL (data analysis)
- Tableau (dashboard)
- Excel (data cleaning)
- draw.io (process flow)
- Google Docs (BRD documentation)
